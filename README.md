# Backend RESTful API for movie platform urTicket using NestJS framework, PostgreSQL, TypeScript, TypeORM, Swagger
[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.


## 1. Installation

### without Docker

```bash
$ npm install
```

### with Docker

```bash
$ docker-compose up
```

## 2. Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## 3. Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
